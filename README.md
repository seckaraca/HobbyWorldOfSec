# 🚀 HobbyWorldOfSec - Profesyonel İş Platformu

Bu proje, **Seckin Karaca** tarafından kurulan ve hobi dünyasını profesyonel bir iş modeline dönüştürmeyi hedefleyen web platformunun teknik altyapısıdır.

## 💼 İş Vizyonu
Bu depo, kurulacak olan web sitesinin kalbi olan veritabanı, API ve sunucu yapılandırmalarını içerir. Projenin amacı, modern teknolojileri kullanarak yüksek performanslı ve güvenilir bir iş platformu sunmaktır.

## 🛠 Teknik Yetkinlikler & Altyapı
Bu profesyonel proje şu teknolojileri içermektedir:
* **Veri Tabanı:** Karmaşık SQL sorgulamaları ve API entegrasyonuna uygun PostgreSQL yapısı.
* **DevOps & CI/CD:** Jenkins ile sürekli entegrasyon ve dağıtım süreçleri.
* **Containerize Uygulamalar:** Docker ve Docker-Compose ile ölçeklenebilir altyapı.
* **Depolama:** Harici harddisk projesini de kapsayan veri yönetimi.
